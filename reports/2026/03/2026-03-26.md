## 🏭 産業・テクノロジー・製造ニュース（メインセクション）

### 日本の産業動向

日本の自動車・半導体産業で複数の構造問題が表面化している。最大の焦点は依然としてネクスペリア問題だ。2025年10月にオランダ政府が安全保障上の理由から中国資本のNexperia（ネクスペリア）を管理下に置いたことで、同社製チップを使用していた日産・ホンダが生産調整を余儀なくされてきた。日産は九州工場でROGUEを中心に2025年11月に2回の生産削減（計2,300台）を実施し、代替サプライヤーの認定には最大6ヶ月が必要と見られている。自動車各社は独自に代替部品の探索・認定作業を進めているものの、半導体の変更には規格認定プロセスが必要なため、迅速な置き換えは難しい状況だ（[Automotive Manufacturing Solutions](https://www.automotivemanufacturingsolutions.com/analysis/nexperia-standoff-imperils-global-auto-production-within-weeks/1590908), [Z2Data](https://www.z2data.com/insights/the-nexperia-chip-crisis-explained)）。

サプライチェーンのセキュリティ対策でも動きがある。経済産業省は2026年度中（10月以降）に「サプライチェーン強化に向けたセキュリティ対策評価制度」を本格運用する方針で、企業がサプライチェーン全体のリスクを可視化・評価するための標準的な枠組みを提供する計画だ（[NRI Secure](https://www.nri-secure.co.jp/blog/security-measures-assessment-system-for-strengthening-the-supply-chain)）。JETROが2026年3月に公表した最新レポートでは、欧米各国が強化するサプライチェーン人権デューデリジェンス法制（EUのCSDD指令、ドイツのサプライチェーン適正評価法等）への対応が日本企業の急務として挙げられており、サプライチェーン管理の法令対応負担は年々増加している（[JETRO](https://www.jetro.go.jp/world/reports/2026/01/35a7f36d5712f4be.html)）。

また対中関係では、中国政府が2026年2月に日本の企業・研究機関40団体を対象にデュアルユース製品の輸出管理リストに追加したことで、レアアース・先端素材の調達多様化が日本企業の急務となっている。さらに2026年3月14日から日本製ハロゲン化ブチルゴムへの中国アンチダンピング税（15〜30.1%）が5年間適用開始となり、自動車部品・医薬品向けの素材サプライチェーンにも影響が出はじめている（[global-scm.com](https://global-scm.com/blog/?p=5280)）。

### グローバル産業動向

製造業のグローバル再編において「リショアリング・パラドクス」が産業界の注目を集めている。2026年のManufacturing Tomorrowの分析（3月）によると、米国への回帰が進む製造現場では「先端設備・自動化ラインは整っているが、それを動かす人材の技術継承とコミュニケーションが機能していない」という状況が多くの工場で起きている。熟練ベテランの退職・多言語化する新規労働者・英語で書かれた工程指示書という三重苦が、設備投資効果を半減させているとの指摘だ（[ManufacturingTomorrow](https://www.manufacturingtomorrow.com/article/2026/03/the-reshoring-paradox-we-built-smart-factories-but-kept-dumb-communication/27049)）。この「最終マイル」の問題を解決するため、製造現場向けAR・多言語AIアシスタント・デジタルツインを活用した「プロフィシェンシー到達時間」の短縮が2026年の製造DXの新たなKPIとして浮上している（[IIOT-World](https://www.iiot-world.com/smart-manufacturing/atoms-bits-neurons-2026-zero-defect-manufacturing-mandate/)）。

物流・サプライチェーン分野では自動化の実装加速が顕著だ。DHL供給チェーン部門は3月17日、SVT RoboticsのSOFTBOT®プラットフォームを全世界の倉庫ネットワークに展開したと発表。従来のカスタムコーディング比で最大12倍速のロボット統合を可能にし、欧州での新規展開を3時間・アジア太平洋での既存ライン追加を無停止で完了した（[DHL Group](https://group.dhl.com/en/media-relations/press-releases/2026/dhl-supply-chain-accelerates-automation-deployments-with-stv-robotics-softbot-platform.html)）。Körberはこの時期、NVIDIAとの戦略的技術連携を発表しており、NVIDIA Omniverseライブラリを活用した物理的精度の高い倉庫デジタルツインの構築と、フィジカルAI（現実世界と相互作用するAI）の物流施設への実装を進めている。3月のIntraLogisteX 2026ではKörberのAI・デジタル部門グローバルヘッドが「予知保全・デジタルツイン・エージェントシステム」を物流DXの三本柱として提示した（[Körber/PRNewswire](https://www.prnewswire.com/news-releases/korber-accelerates-ai-innovation-in-logistics-and-supply-chain-in-collaboration-with-nvidia-302714849.html), [Logistics Manager](https://www.logisticsmanager.com/korbers-michael-becher-to-explore-ai-driven-intralogistics-at-intralogistex-2026/)）。

米国のニューヨーク連銀製造業景気指数（Empire State Manufacturing Index）は3月調査でマイナス圏（−0.2）に落ち込み、出荷・納期の悪化が確認された。一方で設備投資計画は堅調を維持しており、製造業の「ハード投資継続・短期マクロ減速」という二極構造が鮮明になっている（[FinancialContent](https://markets.financialcontent.com/stocks/article/marketminute-2026-3-16-empire-state-manufacturing-hits-a-wall-march-survey-reveals-surprising-contraction-amid-supply-chain-frictions)）。またサプライチェーンダイブの2026年版レポートでは「50%超の企業が調達先・製造拠点の見直しを進めている」とし、「ジャスト・イン・タイムからジャスト・イン・ケースへ」の転換が製造業全体のトレンドとして定着しつつあると分析している（[Supply Chain Dive](https://www.supplychaindive.com/news/supply-chain-risks-trends-outlook-2026/810852/)）。

## 🤖 AI・LLM最新動向

### モデル・プロダクト最新情報

- **Anthropic（Claude Code Auto Mode、3/24）**: Claude Codeに待望の「オートモード」が追加された。これまで開発者はファイル書き込み・bashコマンドのたびに承認を求められるデフォルト動作か、危険な「–dangerously-skip-permissions」モードかを選ぶ必要があったが、Auto Modeはその中間路線を提供する。AIクラシファイヤーが各アクションを事前評価し、安全とみなせば自動実行・リスクがあれば代替アプローチを提示する仕組みだ。大量ファイル削除・機密データ抽出・悪意あるコード実行といる想定外アクションの防止に重点が置かれており、プロンプトインジェクション攻撃のスキャンも組み込まれている。現時点でClaude Sonnet 4.6とOpus 4.6のみ対応し、Teamプラン向けにプレビューが開始、今後Enterprise・APIユーザーへ順次展開される予定（[TechCrunch](https://techcrunch.com/2026/03/24/anthropic-hands-claude-code-more-control-but-keeps-it-on-a-leash/), [Engadget](https://www.engadget.com/ai/anthropic-releases-safer-claude-code-auto-mode-to-avoid-mass-file-deletions-and-other-ai-snafus-142500615.html), [Winbuzzer](https://winbuzzer.com/2026/03/25/anthropic-claude-code-cowork-auto-mode-computer-use-xcxwbn/)）。

- **Microsoft（Agent Framework RC）**: Microsoftが「Microsoft Agent Framework」のリリース候補（RC）版を公開した。PythonとC#（.NET）の両言語に対応するオープンソースのエージェントSDKで、Semantic KernelとAutoGenという従来2系統だった開発基盤を統合した決定版と位置付けられている。MCP（Model Context Protocol）・A2A（Agent to Agent）・OpenAPIをサポートしてベンダー中立性を確保し、逐次実行・並列実行・ハンドオフ・グループチャットといる多様なマルチエージェント構成をストリーミング対応で実装できる。InfoQは「APIは安定済みでGA（正式版）リリースが近い」と報じており、エンタープライズ向けエージェントAI開発の本格的な標準化が進みつつある（[InfoQ](https://www.infoq.com/news/2026/02/ms-agent-framework-rc/), [Microsoft Foundry Blog](https://devblogs.microsoft.com/foundry/introducing-microsoft-agent-framework-the-open-source-engine-for-agentic-ai-apps/)）。

- **NousResearch（Hermes Agent、3/25）**: オープンソースコミュニティで実力派として知られるNousResearchが3月25日、「Hermes Agent」フレームワークを発表。GitHubでホストされ、静的な言語モデルから対話的・自律的なエージェントシステムへとHermesモデルファミリーを進化させるとしている。オープンソースのエージェントフレームワーク競争で注目の新エントリーとなった（[AIToolly](https://aitoolly.com/ai-news/article/2026-03-25-nousresearch-launches-hermes-agent-a-new-intelligent-agent-framework-designed-to-grow-with-users)）。

- **OpenAI（旧バージョンDeep Research廃止、3/26）**: 本日（3/26）をもって、ChatGPTの旧バージョン「Deep Research」モードが廃止された。現行のDeep Research機能は継続提供されるが、過去の会話ログにはアクセス可能。旧モードに依存していたエンタープライズユーザーへの影響が注目される（[Releasebot](https://releasebot.io/updates/openai)）。

### 業界動向・ビジネス

エージェントAIを軸にした産業利用基盤の整備が2026年前半の業界トレンドの中心に据えられている。DEV Communityが3月にまとめた分析では「2026年3月のAI開発ツールは自律エラ（Autonomous Era）への移行を告げる月になった」と総括している。Dapr Agents v1.0（GA）・Microsoft Agent Framework（RC）・NousResearch Hermes Agentというエンタープライズ実用レベルのオープンソースフレームワークが相次いでリリースされたことで、エージェントAIの自社開発コストが大幅に下がり始めた（[DEV Community](https://dev.to/ajay_kumar_1daef5fe089885/ai-developer-tools-enter-autonomous-era-agentic-systems-rise-in-march-2026-1f38)）。製造業でのAIエージェント活用では、物流・在庫管理・品質検査への組み込みが先行しており、Supply Chain Diveの分析によればDescartes・Blue Yonder・Körber・DHLがAIエージェント機能を自社プラットフォームに組み込む競争を本格化させている（[Supply Chain Dive](https://www.supplychaindive.com/news/supply-chain-trends-risks-2026-retail-manufacturing/808797/)）。

### 技術トレンド・ノウハウ

AIコーディング支援の分野では、Claude Code Auto Modeの安全設計アーキテクチャが業界で注目されている。Help Net Securityのレポートでは、同機能がプロンプトインジェクション攻撃対策（観測したコンテンツ内に埋め込まれた悪意のある命令をリアルタイムでスキャン）を搭載していることを評価しつつ、「フォールスポジティブが発生する可能性とトークンコスト・レイテンシの若干の増加はトレードオフとして認識すべき」と指摘している（[Help Net Security](https://www.helpnetsecurity.com/2026/03/25/anthropic-claude-code-auto-mode-feature/)）。並行してオープンソースのエージェントフレームワーク多様化が急速に進んでおり、Microsoft Agent Framework・LangChain Open-SWE・NousResearch Hermes Agentなど3月だけで複数の有力フレームワークが登場した（[AIToolly: Open-SWE](https://aitoolly.com/ai-news/article/2026-03-20-open-swe-a-new-open-source-agent-for-asynchronous-programming-challenges)）。製造・物流AI活用においては、Körber＋NVIDIAが推進するような「NVIDIA Omniverseデジタルツイン＋フィジカルAI」の組み合わせが、製造ラインのシミュレーション→実機展開のギャップを埋める標準手法として浮上しており、工場AIの設計・検証コストを大幅に削減する可能性を持つ。
